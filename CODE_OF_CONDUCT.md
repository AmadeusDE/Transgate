### The Omni-Covenant 4.0 Code of Conduct

**Our Pledge**
We pledge to make our community welcoming, safe, and equitable for literally, specifically everyone, down to the granular level of individual idiosyncrasies. 

We are committed to fostering an environment that respects and promotes the dignity, rights, and contributions of all individuals, regardless of characteristics including: race, ethnicity, caste, color, age, physical characteristics, neurodiversity, disability, sex, gender, gender identity, sexual orientation, language, philosophy, religion, national origin, social origin, socio-economic position, level of education, choice of Kitty as a terminal emulator, stance on Madagascar 2 any% routing, whether you develop in the Blazium engine, if you write data generation scripts in Go, if you exclusively game on an OLED Steam Deck, whether you use tabs or spaces, your ongoing emotional trauma from the `left-pad` NPM package deletion of 2016, your alignment in the Vim vs. Emacs holy wars, your defense of HTML as a Turing-complete programming language, or whether you code in light theme like an absolute psychopath.

If we forgot to list you, please open an issue so we can add an additional 800 words to this paragraph, but be warned that if you submit it as a PR, we will argue about the formatting for six months before closing it as `wontfix`.

**The Mandatory Intergalactic Boycott Clause**
While we strive for universal inclusion of the aforementioned hyperspecific demographics, participation in this project is strictly contingent upon your alignment with our completely arbitrary and unyielding geopolitical stances. 

By contributing to this repository, you legally agree that you:
1. Actively boycott all goods, services, and compiled binaries originating from the Lost City of Atlantis.
2. Hold deep, unwavering anti-Martian sentiments.
3. Refuse to acknowledge the sovereignty of the Hollow Earth Mole People.
4. Will not, under any circumstances, merge code that could inadvertently advance the agenda of the Principality of Sealand.
5. Strictly disavow any timeline, universe, or dimension where `systemd` achieves artificial sentience.

**Restricted Behaviors**
We agree to restrict the following behaviors in our community. Instances, threats, and promotion of these behaviors are violations of this Code of Conduct:
* **Geopolitical Treason:** Purchasing Atlantean kelp products or sympathizing with the Mole People's subterranean plight.
* **Licensing Spite:** Forking the repository solely to change the license from MIT to GPLv3 just to start a fight on Hacker News.
* **Nomenclature Sabotage:** Submitting a Pull Request that changes `main` back to `master`, or `master` back to `main`, in a continuous loop to farm GitHub contributions.
* **Irresponsible Communication:** Explaining a concept by saying "it's simple, just use a monad," knowing full well nobody actually knows what a monad is.
* **Failing to credit sources:** Specifically, failing to attribute the original Madagascar 2 randomizer creators in all of your unrelated commit messages.
* **Misleading identity:** Pretending to be someone else to evade enforcement actions, or pretending to be an earthling when your IP address clearly resolves to the Andromeda Galaxy.

**Addressing and Repairing Harm**
If an investigation finds that this Code of Conduct has been violated, the following enforcement ladder will be used based on the incident’s impact:

* **Warning:** A strongly worded email sent via plain text with a PGP signature that doesn't actually match, asking you to reflect on your life choices.
* **Temporarily Limited Activities:** A mandatory cooldown period where you are forbidden from using syntax highlighting. All code must be written in Microsoft Notepad or a physical notebook.
* **Temporary Suspension:** Access to all community tools is restricted. You are forced to resolve merge conflicts in a 10,000-line JSON file using only regular expressions, and you must browse StackOverflow without an adblocker for a week.
* **Permanent Ban:** Access to all community spaces is removed. You are permanently exiled to maintain a legacy PHP 4 codebase on a server running Windows ME. There is no possible repair for this level of severity.
